PART 2: Automated Firewall Configuration
==============================

1. Open VS Code or Python IDE.

2. Create file:
firewall.py

3. Paste the code:

import requests, subprocess
data = requests.get("https://feodotracker.abuse.ch/downloads/ipblocklist.csv").text.splitlines()
subprocess.run(["netsh","advfirewall","firewall","delete","rule","name=BadIP"])

for line in data:
    if line.startswith("#") or "dst_ip" in line: continue
    ip = line.split(",")[1]
    print("Block:", ip)
    subprocess.run(["netsh","advfirewall","firewall","add","rule",
                    "name=BadIP","dir=out","action=block","remoteip="+ip])

4. Save the file.

5. Open Command Prompt as Administrator.

6. Navigate to file location:

cd C:\Users\YourName\Desktop

7. Check Python:

python --version

8. Install requests library:

python -m pip install requests

9. Run the program:

python firewall.py

10. Verify rules:
Windows Defender Firewall → Outbound Rules → BadIP

Firewall rules were successfully created manually and automatically.

LAB- PASSWORD CHECKER

import re

p = input("Enter password: ")

if len(p) < 8:
    print("Weak: Minimum 8 characters")
elif not any(c.isdigit() for c in p):
    print("Weak: Need a number")
elif not any(c.isupper() for c in p):
    print("Weak: Need uppercase")
elif not any(c.islower() for c in p):
    print("Weak: Need lowercase")
elif not re.search(r"[!@#$%^&*]", p):
    print("Medium: Add special character")
else:
    print("Strong Password")
